Blackjack

CLASS JAVA PROJECT

  • Role

    Java developer

  • Used

    Java

  • URL

    https://github.com/mimi-miyeon/blackjack

당근마켓에서 영어 재능을 기부하고 Java를 배울 수 있는 재능교환 기회를 갖게 되었다. 이 기회로 생소했던 클래스에 대한 개념에 조금 친숙해질 수 있었다.

무작위로 카드를 뽑는 'randomSingleton' 클래스와 게임을 세팅하고 동작하게 하는 'Blackjack' 클래스로 작업했다. 게임을 초기에 세팅하고 게임을 진행하고 게임이 끝나면 초기화하는 동작에서 겹치는 코드들을 최대한 함수로 묶으려고 노력했다.

재능기부를 해주신 분이 알고리즘 쪽 경력자라서 디자인에는 시간을 할애하지 않았다. 주로 디자인 위주로 일해왔던 나에게는 완전하게 알고리즘에만 집중할 수 있어서 개발자가 되기 위해 부족한 부분을 보완할 수 있었다.

단순하게 Java 공부를 하는 시간이 아닌 리팩토링 등 코딩하는 데 도움 되는 조언을 많이 얻을 수 있는 시간이었다. 이 재능교환으로 풀스택이 되고자 하는 마음을 확고하게 갖게 되는 기회가 되었고, 취업 이후 기회가 된다면 다시 계속해서 진행할 예정이다.
