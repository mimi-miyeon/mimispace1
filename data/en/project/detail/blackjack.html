I worked on a 'randomSingleton' class for randomly drawing cards and a 'Blackjack' class for setting up and running the game. I tried to minimize redundant code by grouping common actions into functions, such as setting up the game initially, progressing the game, and resetting it when the game ends.

This project focuses solely on algorithm development without any screen design.
