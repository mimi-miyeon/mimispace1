A service introduction website for the RealSeller service. This project involved redesigning the existing company website, splitting it into separate sections for service introduction and company information.

RAS (Real Authentication Service) is the authentication method used by RealSeller, allowing users to search for verified sellers. To save time, a CSS utility class-based template was purchased and used for the design, focusing on clearly conveying the core features and usage of the service.
